To load the *.lib file:

Download the *.lib file. Start KiCAD and select either "Symbol Editor" or "Tools" -> "Edit Schematic Symbols";
select "File" -> "Add Library..." and point to the location where you downloaded the *.lib file.

To load the footprint:

Download the *.pretty folder with the *.kicad_mod mod inside. Start KiCAD and select either "Footprint Editor" or
"Tools" -> "Edit PCB Footprints". Locate and select the *.pretty folder on your computer.
